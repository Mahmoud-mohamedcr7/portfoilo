# Mahmoud Mohamed — Portfolio

## Run
- Open `index.html` in your browser.

## Customize
- **Projects**: edit cards + links in `index.html` under the `#projects` section.
- **Images**: replace files in `assets/img/` with your screenshots.
- **Theme**: tweak colors in `assets/css/style.css` (`:root` variables).

## Notes
- The **Contact form** uses `mailto:` (no backend). If you want real sending, I can add a PHP handler or a Formspree/Web3Forms integration.
- The **CV download** points to `assets/Mahmoud_Mohamed_CV.html`. Use “Print / Save as PDF” to export a PDF.
